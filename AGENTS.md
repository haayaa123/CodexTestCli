# AGENTS.md

## Project overview
Vite + Vue 3 のサンプルアプリです。

## Setup / Dev / Build
- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`

## Expectations
- 変更後は必ず `npm run build` を実行して成功ログを提示すること
- 既存の構成（Vite/Vue標準）を崩さず、必要最小限の変更にすること

## Code style
- 既存の ESLint/Prettier 設定に従う（導入済みなら）
